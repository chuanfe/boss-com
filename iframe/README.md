## Iframe iframe框架

### 属性配置
------
| 属性名 | 类型 | 默认值 | 描述 | 备注 |
| ------ | ------ | ------ | ------ | ------ |
| iframe地址 | Text | http://www.baidu.com |设置iframe外链的src地址 | - |

### 数据配置
------

#### source（默认）

##### 数据字段

| 字段名 | 类型 | 默认值 | 描述 | 备注 |
| ------ | ------ | ------ | ------ | ----- |
| url | String | '' |iframe外链的`src`地址 | - |

##### 静态数据
```
[
  {
    "url": ""
  }
]
```

### 数据配置
------

#### source（默认）

##### 数据字段

| 字段名 | 类型 | 默认值 | 描述 | 备注 |
| ------ | ------ | ------ | ------ | ----- |
| x | String | - | 散点类目 即x轴的值 | - |
| y | String | - | 散点值 即y轴的值 | - |
| s | String | - | 系列值 数据系列中系列名 | - |
